Errores comunes en un algoritmo
Impresion: es cuando el usuario puede confundir una variable con texto ocasionando el error.
Peticion: es cuando el usuario pide alguna cosa y en la parte donde lo pide escribe mal la variable y se genera el problema.
Declaracion: es cuando en una variable ponemos alguna palabra y en vez de poner las dos comillas dobles ( "" ) ponen las normales ( '' ).
Modificacion: es donde se cambia el nombre del valor de la variable y se crea un error modificando cosas donde no deben de ir.
